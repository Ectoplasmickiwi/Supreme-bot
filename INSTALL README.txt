Here are the requirements you need to meet when hosting this bot,
- MySQL Database
- Discord Bot Token (visit developers.discord.net)
- Node.js (version 10+)

How to install this bot,
- Insert your bot token from the developers discord website in config.json
- Insert your database credentials in config.json (to store accounts/info)
- Run 'npm install' to fully install the bot in the command line.
- Start the bot by using 'node index.js'

How to edit your bot:
- You can edit your bot gens and roles in the config.json or use the commands in discord.
- Use '-help' for a full list of commands/functions
